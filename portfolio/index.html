Portfolio

  • Scoren met Sporen
    My graduation project for ProRail. This game was created by a team of 12 people (10 programmers, 2 artists). It was made with Monogame as its framework, using C#. We used the Scrum approach to have every member of the team be able to contribute properly.
    This game translates a boardgame to a digital format. It is used internally by ProRail to teach new employees about the responsibilities of the company. My biggest responsibility during the project was the in-game level editor. During this project I learned a lot about the importance of high code quality. Whenever the project lacked in this aspect, making changes to the game's logic became impossible. Thus, the code quality was held in high regard, which is something that has stuck with me since.
    Trigger warning: This trailer filled with dank memes, but does show off the game's features. It is the best footage I have of it for now.


  • Testzone (github link)
    A game that was made during my Game Design minor. The theme was 'Design for space', in which spatial challenges were the goal of the design. It's a online free-for-all game in which four player's are pitched against each other in a test facility for physics-based gadgets. It was made using C# in Unity3D and the build-in UNet networking. I was the lead (and sole) programmer of the project, in addition to assisting in Game Design. During this project I learned a lot about Unity. Looking back at it, I'm not entirely pleased with the code quality. I feel I have improved tremendously in this aspect since.


  • Megamen
    The introduction project during my first year of my bachelor. It was made using C# in XNA and the Lidgren networking library. During this project I was eager to learn programming, which resulted in me taking on a lot of responsibilities. I implemented the character controls, enemy AI and online co-op. I was also responsible for the sound design, creating all SFX and the OST myself using FL Studio.
